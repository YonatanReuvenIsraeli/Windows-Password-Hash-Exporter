# Windows Password Hash Exporter

## This batch file, named "Windows Password Hash Exporter", is designed for educational purposes only to demonstrate potential vulnerabilities in Windows operating systems. It should only be used in controlled testing environments with explicit authorization.

## Notes:
**1.** You will need to use impacket-secretsdump in Kali Linux to view the hashes.  
**2.** Once you have the hash, you can use a tool like Hashcat to crack the hash.
